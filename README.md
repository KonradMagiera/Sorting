# Sorting
Sorting algorithms with visualisation


## Built With

* [Java](https://java.com/)
* JavaFX

## Version

0.7

## Algorithms

There are available several algorithms:
* Bubble sort
* Comb sort
* Cycle sort
* Insertion sort
* Merge sort
* Recursive bubble sort
* Recursive insertion sort
* Selection sort
* Shell sort
* Quick sort
